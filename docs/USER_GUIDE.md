# Media Player 用户指南

## 快速开始

### 启动应用

1. **启动后端服务**
   ```bash
   cd /home/admin/media-player
   cd backend && source .venv/bin/activate
   python3 -m uvicorn backend.main:app --host 0.0.0.0 --port 8000 --app-dir ..
   ```

2. **启动前端开发服务器** (可选，开发模式)
   ```bash
   cd /home/admin/media-player/frontend
   npm run dev
   ```

3. **访问应用**
   - 开发模式: http://localhost:5173
   - 生产模式: http://localhost:8000

## 核心功能

### 🔍 搜索媒体

在顶部搜索栏输入关键词，支持搜索:
- **YouTube** - 全球最大的视频平台
- **B站** - 中文视频社区
- **网易云** - 音乐流媒体

搜索结果会以卡片形式展示，包含:
- 缩略图
- 标题
- 时长
- 来源平台

### 🎵 播放控制

#### 主播放器
- 播放/暂停
- 进度条拖动
- 音量调节
- 全屏模式
- 倍速播放

#### 迷你播放器
- 底部固定显示
- 显示当前播放信息
- 快捷控制按钮

#### 播放模式
- 🔀 随机播放
- 🔁 列表循环
- 🔂 单曲循环
- ➡️ 顺序播放

### 📋 播放列表

左侧面板显示当前播放列表:
- 拖拽排序
- 删除歌曲
- 清空列表

### ❤️ 收藏功能

需要登录后使用:
- 点击收藏按钮添加
- 在收藏面板管理
- 云端同步

### 📜 播放历史

自动记录播放历史:
- 最多保留 100 条
- 记录播放位置
- 云端同步 (登录后)

### 🎤 歌词显示

播放音乐时自动获取歌词:
- 实时同步
- 滚动显示
- 支持手动滚动

## 常见问题

### Q: 视频无法播放？

**可能原因:**
1. 视频来源限制 (如地区限制)
2. 网络连接问题
3. 视频格式不支持

**解决方法:**
- 尝试其他来源
- 检查网络连接
- 刷新页面重试

### Q: 搜索结果加载慢？

**可能原因:**
- 网络延迟
- 搜索源服务器响应慢

**解决方法:**
- 缩小搜索范围 (选择单个源)
- 等待缓存生效 (5 分钟内相同搜索会更快)

### Q: 登录后数据没有同步？

**解决方法:**
1. 检查网络连接
2. 刷新页面
3. 重新登录

### Q: 如何清除本地数据？

打开浏览器开发者工具:
1. Application → IndexedDB → 删除相关数据库
2. Application → Local Storage → 清除相关数据
3. 刷新页面
