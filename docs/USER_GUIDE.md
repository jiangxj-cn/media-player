# Media Player 用户指南

## 快速开始

### 访问播放器

打开浏览器，访问 http://localhost:8000（或您的服务器地址）

### 基本操作

#### 1. 搜索媒体

- 在顶部搜索框输入关键词
- 选择搜索源：
  - **YouTube**: 视频搜索
  - **B站**: 哔哩哔哩视频
  - **网易云**: 网易云音乐
  - **全部**: 同时搜索所有源

#### 2. 播放视频/音乐

- 点击搜索结果中的项目开始播放
- 播放器支持：
  - 播放/暂停
  - 进度拖动
  - 音量调节
  - 全屏模式
  - 倍速播放

#### 3. 播放列表管理

- 点击右侧"播放列表"图标
- 创建新列表
- 拖拽排序
- 一键清空

#### 4. 收藏功能

- 点击♥图标收藏
- 在收藏面板查看
- 快速添加到播放列表

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Space` | 播放/暂停 |
| `←` | 后退 5 秒 |
| `→` | 前进 5 秒 |
| `↑` | 音量增加 |
| `↓` | 音量减少 |
| `M` | 静音切换 |
| `F` | 全屏切换 |
| `R` | 随机播放 |
| `L` | 循环模式 |
| `/` | 聚焦搜索框 |
| `Esc` | 关闭面板 |

## 播放模式

### 顺序播放
按列表顺序依次播放

### 随机播放
随机选择下一个播放

### 单曲循环
重复播放当前项目

### 列表循环
播放完最后一首后返回第一首

## 用户账户

### 注册

1. 点击右上角"登录"按钮
2. 选择"注册"标签
3. 填写用户名、邮箱、密码
4. 点击注册

### 登录

1. 点击"登录"
2. 输入用户名和密码
3. 登录后同步数据

## 常见问题

### Q: 视频无法播放？

1. 检查网络连接
2. 尝试其他视频源
3. 刷新页面重试

### Q: 搜索结果为空？

1. 检查关键词拼写
2. 尝试简化关键词
3. 切换搜索源

### Q: 歌词不同步？

- 部分歌曲可能没有官方歌词
- 可以手动调整偏移

### Q: 如何下载？

- 点击分享/下载按钮
- 选择下载格式
- 注意版权限制
